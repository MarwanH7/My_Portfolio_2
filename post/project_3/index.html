Introduction

Predicting flight delays has always been an interesting and challenging topic for Data Scientists. The problem of flight delays costs both airline operations and consumers a lot of money and time. Flight delay prediction plays a key role in the aviation industry. Much research has been dedicated to developing and deploying various Machine Learning models to increase the accuracy of predictions. In this one-week Midterm Project, our goal is to tackle this challenging problem using the data science tools we’ve learned and practiced throughout the first half of our boot camp. We hope to learn and gain experience from this real-world problem.

Work Done - Flight Delay Prediction Estimator

  • Created a Machine Learning model that estimates flight delays.
  • Wrote SQL queries in VSCode to export a sample of the data stored in a PostgreSQL database.
  • Engineered features, including weather conditions (using a weather API), route traffic, and aggregated datetime dates.
  • Optimized our linear regression model’s performance using ensemble stacking techniques to obtain the best model.
  • In the process of building a user-facing API using Streamlit.
